# SmartPortfolio

智能投资组合管理应用 - 帮助用户管理基金/股票持仓，应用经典投资策略，提供智能调仓建议。

## 功能特性

- 📊 **仪表盘** - 资产总览、今日收益、持仓分布可视化
- 📈 **策略分析** - 雷达图对比当前持仓与策略目标，偏离度分析
- ⚖️ **调仓建议** - 自动生成买卖建议，调仓前后对比

## 支持的投资策略

- 哈利·布朗永久组合 (25/25/25/25)
- 桥水全天候策略 (30/40/15/15)
- 防御网格 (20/60/10/10)
- 自定义策略

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts (图表库)
- React Router (路由)
- Lucide React (图标)

## 快速开始

### 前置要求

- Node.js 18+ 和 npm/pnpm/yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看应用

### 构建生产版本

```bash
npm run build
```

## 页面说明

### 仪表盘
- 显示总资产和今日收益
- 选择投资策略
- 环形图展示11只基金的持仓分布

### 策略分析
- 雷达图对比当前持仓与策略目标
- 偏离度列表，用颜色标记风险等级
- 查看调仓方案入口

### 调仓建议
- 调仓摘要（交易总额、预估费用等）
- 买卖操作列表
- 调仓前后雷达图对比
- 确认调仓按钮

## 注意事项

- 当前版本使用模拟数据
- 暂无后端连接
- 暂无数据持久化
